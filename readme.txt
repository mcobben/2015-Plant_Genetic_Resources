Here are the occurrence files for all the Lactuca species investigated.

All files contain the species name (col1), longitude and latitude of the collection location (col2 and col3),
and the values of the nineteen bioclimatic variables at this location.

The files in which the species name is followed by a capitalized code are segments of the complete data 
to study the species in the specific region (EUR for Eurasia, GRC for Greece, etc.)
